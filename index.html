Data Types

UFO

  • row datatype: items
  • 'datetime' column datatype: attributes
    • attribute type: ordered (ordinal and cyclical)
  • 'city' column datatype: attributes
    • attribute type: categorical
  • 'state' column datatype: attributes
    • attribute type: categorical
  • 'country' column datatype: attributes
    • attribute type: categorical
  • 'shape' column datatype: attributes
    • attribute type: categorical
  • 'duration (seconds)' column datatype: attributes
    • attribute type: ordered (quantitative and sequential)

Airports

  • row datatype: items
  • 'AIRPORT_NAME' column datatype: attributes
    • attribute type: categorical
  • 'LATITUDE' column datatype: position
  • 'LONGITUDE' column datatype: position

Google-web

  • row datatype: link
  • 'FromNodeId' column datatype: node (quantitative)
  • 'ToNodeId' column datatype: node (quantitative)
